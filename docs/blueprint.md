# **App Name**: Mutantes Refugiados Sheet

## Core Features:

- Character Information Display: Displays basic character information, including name, player name, mutation type, stage, and experience points.
- Attribute Radar Chart: Presents attributes in a radar chart, allowing users to view and modify attribute levels.
- Skill Management: Implements skill management with dynamic addition, customizable names, and attribute association.
- Derived Statistics Calculation: Calculates and displays derived statistics (Movement, Parry, Toughness) based on character attributes and skills.
- Wound Tracking: Interactive display for tracking character wounds with clickable skull icons.
- Fatigue Tracking: Interactive display for tracking character fatigue levels with clickable spiral icons.
- Edges and Hindrances Selection: Enables the addition of edges/hindrances from a pre-populated list, with descriptions shown on hover.
- Equipment Management: Enables the addition of equipment from a pre-populated list, with statistics shown on hover.

## Style Guidelines:

- Primary color: Deep crimson (#800020), evoking the grit and danger faced by the refugee mutants.
- Background color: Dark grey (#222222), providing high contrast for readability and accentuating the somber setting.
- Accent color: Electric purple (#BF00FF), symbolizing mutation and unique abilities.
- Body and headline font: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body. Modern and readable, enhancing the technical aspect of the sheet.
- Use clear, minimalist icons for attributes and skills to improve usability.
- Organize the character sheet into distinct sections (Information, Attributes, Skills, etc.) using a tabbed interface for easy navigation.
- Use subtle animations when updating statistics, such as the radar chart smoothly updating as attribute points are added.
